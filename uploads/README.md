# Campaign Images

This directory contains images for charity campaigns.

## Image Naming Convention
Images should be named following this pattern:
`campaign-XXX-description.jpg`

Example:
- `campaign-001-tet-an-lac-mang-xuan-ve-nha.jpg`
- `campaign-002-thap-sang-uoc-mo-hoc-bong-cho-em.jpg`

## Image Requirements
- Format: JPG, PNG
- Recommended size: 800x600px or larger
- Aspect ratio: 4:3 or 16:9
- File size: < 2MB for optimal web performance
